# Deep Learning in Biology @ HSE FCS

Курс по применению методов **deep** learning в биологии и медицине для студентов ФКН НИУ ВШЭ. [hse](https://www.hse.ru/en/edu/courses/646470273)

***

## Структура репозитория

Репозиторий организован по типам материалов:

- `lectures/` — pdf‑файлы лекций, общий README по лекциям.
- `seminars/` — ноутбуки и код семинаров, для некоторых тем будут отдельные подпапки с дополнительными материалами.
- `homeworks/` — формулировки домашних заданий и вспомогательные материалы.

***

## Основная информация о курсе

- Университет: НИУ ВШЭ, ФКН. [hse](https://www.hse.ru/en/ba/compds/documents)

### Блоки курса

Курс состоит из трёх взаимосвязанных блоков:

1. Биоинформатика и DL в биологии (Андрей Боревский @bor_loco).
2. Physically Informed Neural Networks (Александр Тараканов).  
3. ИИ и новые материалы (Михаил Лазарев).  

Каждый блок имеет собственные акценты в части методов и приложений, но общая линия курса — современные DL‑подходы к анализу биологических и материаловедческих данных. [hse](https://www.hse.ru/en/edu/courses/646470273)
